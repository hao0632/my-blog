## Markdown 写作指南

这篇文章教你如何使用 Markdown 语法。

### 标题

使用 `#` 来创建标题，从 `#` 到 `######` 分别对应 h1 到 h6。

### 文本格式

- **粗体** 使用 `**粗体**`
- *斜体* 使用 `*斜体*`
- ~~删除线~~ 使用 `~~删除线~~`

### 链接和图片

[访问我的 GitHub](https://github.com)

### 代码块

行内代码使用 `code`

代码块：

```python
def hello():
    print("Hello, World!")
```

### 列表

无序列表：

- 项目一
- 项目二
  - 子项目
  - 子项目

有序列表：

1. 第一步
2. 第二步
3. 第三步

### 引用

> 这是一段引用文字。
> 可以有多行。

### 表格

| 列1  | 列2  | 列3  |
| --- | --- | --- |
| 数据1 | 数据2 | 数据3 |
| 数据4 | 数据5 | 数据6 |
